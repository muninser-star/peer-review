# Взаимная оценка продактов (Peer Review)

Одностраничное веб-приложение для взаимной оценки продакт-менеджеров.
Каждый продакт выбирает своё имя, оценивает коллег и отправляет результат
в Google-таблицу одной кнопкой.

## Как устроено

- `index.html` — сама страница. Полностью автономна (без бэкенда), данные
  заполнения хранятся в браузере (localStorage) до отправки.
- `apps-script.gs` — код Google Apps Script, который принимает ответы
  и пишет их строками в Google-таблицу.

## Вопросы оценки

1. Насколько комфортно работать с коллегой как с продактом — оценка ⭐ 1–10
2. Насколько он соответствует своей роли
3. Сильные стороны
4. Слабые стороны / зоны роста
5. Что стоит улучшить

## Настройка интеграции с таблицей

1. Создать Google-таблицу → **Расширения → Apps Script**.
2. Вставить код из `apps-script.gs`, сохранить.
3. **Развернуть → Новое развёртывание → Веб-приложение**, доступ «Все».
4. Скопировать URL веб-приложения и вставить его в `index.html`
   в строку `const SCRIPT_URL = "..."`.
